개발자들끼리 소스코드 리뷰등을 통해 소스코드를 검토하지만 "문서화 리뷰"는 대게 하지 않는다.
하지만 제품 개발이 완료되고 유지보수할 때 문서화된 자료가 아무것도 없다면 이는 유지보수에 큰 어려움이다.
따라서 많이 사용하는 클래스, 함수는 스핑크스를 통해 문서화한다면 유지보수에 좋을 것이다.
